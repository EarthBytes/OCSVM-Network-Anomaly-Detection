# OCSVM-Network-Anomaly-Detection

Network anomaly detection using One-Class SVM to identify unusual network activity in traffic data.

## Setup Requirements and Dataset

```bash
pip3 install -r requirements.txt
```

```bash
python3 -m src.main
```

### Dataset Source

https://www.kaggle.com/datasets/ziya07/network-traffic-anomaly-detection-dataset/data
